# Event Detail Options Displayed

### 

## Javascript Code
```js
window.appEventData = window.appEventData || [];
appEventData.push({
  "event": "Event Detail Options Displayed",
    "eventDetailOptions": {
        "optionList": [
            {
                "event": {
                    "eventId": "<eventId>",
                    "eventName": "<eventName>"
                }
            }
        ]
    }
});
```
